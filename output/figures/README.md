# Figure Manifest

These figures are generated by `scripts/plot_publication_figures.py` from the
committed CSV artifacts under `output/`. They are intended as visual summaries
of the same evidence reported in `report/report.md` and
`report/methodology.md`.

| Figure | Source data | Purpose |
|---|---|---|
| `01_predictive_power_vs_baselines.png` | `output/subminute_decay/curve_data.csv` | Shows cross-asset OFI predictive power against own-OFI and return-history baselines by bar width. |
| `02_cross_edge_decay.png` | `output/subminute_decay/curve_data.csv` | Shows the R2 edge of cross-asset OFI over each baseline and how it decays. |
| `03_corrected_pass_heatmap.png` | `output/subminute_decay/significance.csv` | Shows how many corrected controls each bar-width/scheme cell passes. |
| `04_all_control_pass_counts.png` | `output/subminute_decay/significance.csv` | Counts schemes passing all corrected controls by bar width. |
| `05_robustness_pass_summary.png` | `output/robustness_300s/significance.csv`, `output/subminute/significance.csv`, `output/subminute_decay/significance.csv` | Compares corrected pass rates across the 300s candidate, 1s/5s/10s grid and 1s-30s decay grid. |
| `06_cross_impact_matrix.png` | `output/core/matrix.csv` | Visualises the descriptive 60s cross-impact coefficient matrix. |
| `07_data_depth_diagnostics.png` | `output/core/coverage.csv`, `output/core/pca.csv` | Summarises spread levels and PCA depth-factor stability. |
| `08_research_workflow.png` | Methodology structure | Shows the reproducible workflow from raw OKX data to final interpretation. |
